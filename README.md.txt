# site_im100ruv

site_im100ruv : my website with portfolio and blogs

This project is done using Python and Django framework.





Steps to run application



1)Download all files and folder (or simply clone).
2)Make sure you have Python and Django installed on your system.
	If not, search for a guide to install Python and Django. 

3)Go to the root directory in your command prompt and type following:

	python manage.py migrate

	python manage.py runserver

4)Open localhost:8000 in browser.
